# 微网数电页面讲解稿

## Talk Objective
让面试官记住三件事：
1. 我能先看懂一家公司公开到底在做什么，而不是先套一个分析框架。
2. 我会把零散公开信息拉成同一口径，而不是停在资料堆砌。
3. 我能给一个不过度的初步判断，而不是上来就装作模型很完整。

## 2-Minute Version
先说明一下，我提前做了一个很短的梳理，主要想展示三步：先看公司公开在做什么，再把公开资料按同一口径整理，最后给一个很轻的机会判断。

我对微网数电的理解是，它不是单纯卖充电设备，也不是单纯做软件平台，而是围绕绿色出行补能场景做解决方案。公开最明确的两条主线，是两轮换电和聚合超充；背后又结合了电池服务、BaaS、光储充和数字化运营这些能力。

所以我会把它理解成：前台是两类业务，后台是一套能力。再结合比亚迪财报口径看，它又不是比亚迪全资子公司，而是联营企业，这也说明它更像一个有生态协同属性的场景化公司。

我做这页想展示的，不是我已经很懂它内部情况，而是如果让我做这个岗位，我会先把公司定位、业务主线和生态关系讲清楚；再把行业、政策、项目这些公开信息整理成统一口径；最后才去做机会识别和初步评估。

## 3-Minute Version
先说明一下，这页不是在猜公司内部数据，而是我基于官网、财报和公开资料，做一版公司公开业务梳理。

我对微网数电的理解是，它不是单纯卖充电设备，也不是单纯做软件平台，而是围绕绿色出行补能场景做解决方案。公开最清楚的前台业务有两条，一条是两轮换电，一条是聚合超充。

再往后看，官网还提到电池银行开放平台、数字电池服务、光储充一体场景。更合理的理解是：前台是换电和超充，后台是电池服务、场站能源管理和调度能力。

治理结构上，它也不是比亚迪全资体系。比亚迪财报口径里，它是联营企业，而且 2021 年设立时是多方出资。所以我会把它看成一个有明显生态协同属性的补能解决方案公司。

如果基于这个理解再往下做商业分析，我会先把公开信息整理成一张统一口径的底表。比如公司公开信息里看业务场景和治理结构，行业和政策信息里看市场阶段和落地约束，项目和案例信息里看客户类型、收入逻辑和关键变量。

在这一步做完以后，我才会去做机会初筛，先判断需求是不是真的存在，再看价值落点和收入逻辑，最后看最大的落地约束是什么。输出不是假装很完整的模型，而是先给出值得继续看、需要补数据还是暂不优先的初步判断。

所以这页最想展示的，其实是我的商业洞察方式：先看懂公司，再把信息整理成统一口径，最后支持后面的机会判断。
