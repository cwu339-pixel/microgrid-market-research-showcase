# 微网数电精简讲法

## 最自然的开场
“我提前基于公开信息，做了一版很短的梳理。主要不是想假装知道公司内部情况，而是想展示一下，如果让我做这个岗位，我会怎么先看懂一家公司公开到底在做什么。”

## 一分钟半版本
“我对微网数电的理解是，它不是单纯卖充电设备，也不是单纯做软件平台，而是围绕绿色出行补能场景做解决方案。  

公开最明确的两条主线，一条是两轮换电，另一条是聚合超充。再往后看，官网还提到 BaaS、电池服务、光储充和数字化运营这些能力。所以我不会把它讲成四个平行赛道，而会理解成：前台是两类业务，后台是一套能力。  

再结合比亚迪财报口径看，它又不是比亚迪全资子公司，而是联营企业，这说明它有明显的生态协同属性。  

我做这版梳理想展示的，其实就是这件事：我会先把公司定位、业务主线和生态关系理清楚，再决定后面应该怎么继续做商业分析。”  

## 两分钟版本
“我提前基于公开信息做了一版很短的梳理，主要不是想证明我对行业懂得有多深，而是想展示一下，如果让我做这个岗位，我会怎么先看懂一家公司。  

我对微网数电目前的理解是，它不是单纯卖充电设备，也不是单纯做软件平台，而是围绕绿色出行补能场景做解决方案。公开信息里最明确的两条主线，一条是两轮换电，一条是聚合超充。  

再往后看，官网还提到电池银行开放平台、数字电池服务、光储充和数字化运营这些能力。这里我不会直接把它们写成平行赛道，因为更合理的理解是：前台是换电和超充，后台是电池服务、场站能源管理和调度能力。  

治理结构上，它也不是比亚迪全资体系。比亚迪财报口径里，它是联营企业，而且 2021 年设立时是多方出资。所以我会把它看成一个有生态协同属性的补能解决方案公司。  

我做这版梳理，想展示的就是这件事：我会先把一家公司的公开业务、生态位置和可能的方向理清楚，再决定后面怎么继续做研究和分析。”  

## 现场怎么点
1. 先开 `live-walkthrough.html`
2. 只讲这几个区块：
   - `总览`
   - `公司定位`
   - `业务主线`
   - `判断依据`
   - `我的理解`
3. 不要逐条念 bullet
4. 不要主动展开来源页

## 你讲的时候记住的三句
1. “先把公司到底做什么讲准。”
2. “前台是两类业务，后台是一套能力。”
3. “我想展示的是，我能比较快地看懂一家公司的公开业务。”

## 最后收口
“所以这版内容不是想替代内部资料，而是想展示，如果让我做这个岗位，我会怎么把公开信息先整理成结构化的公司理解和商业判断。”
